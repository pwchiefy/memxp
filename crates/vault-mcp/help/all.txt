memxp - Your Persistent Memory

WORKFLOW:
1. whats_saved() - See what's in your memory
2. find(query) - Search for something saved
3. recall(path) - Look up a specific item
4. remember(path, value, ...) - Save something for later
5. read_instructions(name) - Read a saved how-to guide

SECURITY:
- Values are MASKED in search results
- Use redact=true to copy to clipboard without exposing in chat
- vault_inject() sets env vars without returning values
- All access is logged via vault_audit()

TOOLS:
