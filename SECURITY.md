# Security Policy

memxp is a security-sensitive project. Please do not report vulnerabilities through
public issues.

## Reporting security issues

Report security issues privately to:

- dubious_miner93@icloud.com
- or use GitHub private reporting on the repository

Include:

- version/commit hash
- steps to reproduce
- potential impact and blast radius
- any proof-of-concept artifacts
- suggested prompt to fix it

## Response expectations

- Acknowledgement: within 72 hours
- Status update: within 7 days
- Fix timeline: prioritized based on severity

## Responsible disclosure

- Do not access, modify, or destroy data outside your authorized test scope.
- Avoid public discussion until a patch is released.
